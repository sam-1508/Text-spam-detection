# Text-spam-detection
Here is a way to detect spam messages by providing your text
**Libraries needed**

Use the "pip install package-name" command to install the following: 
1. pandas
2. matplotlib
3. nltk
4. seaborn
5. streamlit
6. pickle

**Training  the model:**
Run the "email spam detection.ipynb" notebook
